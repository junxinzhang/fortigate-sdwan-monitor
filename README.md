## About

This repository contains the landing page for **FortiGate SD-WAN Health Monitor** - an enterprise network monitoring solution.

### Key Features Showcased

- **Real-time Health Monitoring** - Track latency, jitter, packet loss, bandwidth
- **Smart Threshold Alerts** - Customizable alerts with cooldown mechanism
- **Multi-channel Notifications** - Email, Microsoft Graph API, Webhook
- **Historical Trend Analysis** - 6h/24h/7d visualization
- **Web Dashboard** - Dark theme UI with auto-refresh

## Tech Stack

This landing page is built with:

- HTML5 / CSS3
- Tailwind CSS
- AOS (Animate On Scroll)
- Remix Icon
